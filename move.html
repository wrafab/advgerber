Crickit and Circuit Playground


The Crickit and Circuit Playground is actually two low cost devices that allow you to easily program motors, sensors, servos, solenoids, leds and much more. We will use them to learn how to use the sensors, servos, capacitive touch, solenoids, leds and speakers to create a project.

Assignment

You need to create a project using the Crickit and Circuit Playground. The project must have at least two of these: motor, servo, speaker, led, capacitive touch, solenoid, sensor or input. You need to also create it using other tools in the WIC. The project should be innovative (no copying a project online). It must be original. You will need to include photos of the project and all files used to create it.
